Номер “Комфорт” с видом на горы - идеальное решение для тех, кто хочет насладиться красотой природы и уютом отеля. Этот номер оснащен всем необходимым для комфортного проживания: удобной кроватью, шкафом,прикроватными тумбами, холодильником.
